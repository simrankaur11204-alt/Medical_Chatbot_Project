# AI Medical Chatbot

This project is a Final Year AI-based Medical Chatbot built using:

- Python
- Flask
- NLP (TF-IDF + Cosine Similarity)
- SQLite Database
- HTML, CSS, JavaScript

## Features

- Accepts user symptoms
- Matches symptoms using NLP
- Provides medical suggestions
- Stores chat history in SQLite database
- Web-based interface

## Technologies Used

- Python
- Flask
- Scikit-learn
- SQLite
- HTML/CSS

## Disclaimer

This chatbot provides general health information only.
It is not a replacement for professional medical advice.
